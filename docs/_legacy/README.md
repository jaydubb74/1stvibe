# v1 Documentation (Archived)

This directory holds the v1 1stvibe documentation, preserved for reference during the v2 rebuild.

## What's here

- `INDEX.md` — v1 documentation index (outdated)
- `architecture/` — v1 system overview, API routes, database schema
- `features/` — v1 feature docs (demo generator, tutorial system, homepage UX, auth, etc.)
- `guides/` — v1 style guide, code standards, documentation standards, code review checklist
- `strategy/` — v1 vision, customer profile, brand kit, brand voice

## What v2 uses from these

Nothing directly. But the brand-identity content from `strategy/brand-kit.md`, `strategy/brand-voice.md`, and `guides/style-guide.md` has been consolidated into the v2 canonical source: [`/docs/rebuild/brand-and-style.md`](../rebuild/brand-and-style.md).

These originals remain here for traceability — what changed and why is visible in the git history of the consolidation.
